# Article L2223-36

Les personnes morales peuvent être déclarées responsables pénalement, dans les conditions prévues par l'article 121-2 du code
pénal, des infractions définies à l'article L. 2223-35. 

Les peines encourues par les personnes morales sont : 

1° L'amende, suivant les modalités prévues par l'article 131-38 du code pénal ; 

2° Les peines mentionnées aux 2°, 3°, 4°, 5°, 6°, 7°, 8° et 9° de l'article 131-39 du code pénal ; l'interdiction mentionnée
au 2° de l'article 131-39 du même code porte sur l'activité dans l'exercice ou à l'occasion de l'exercice de laquelle
l'infraction a été commise.
